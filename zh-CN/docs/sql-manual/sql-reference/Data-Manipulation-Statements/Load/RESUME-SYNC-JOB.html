    RESUME-SYNC-JOB

    # RESUME-SYNC-JOB

    # Name

    RESUME SYNC JOB

    # Description

    通过 job_name恢复一个当前数据库已被暂停的常驻数据同步作业，作业将从上一次被暂停前最新的位置继续同步数据。

    语法:

    RESUME SYNC JOB [db.]job_name
    1

    # Example

    1. 恢复名称为 job_name 的数据同步作业

      RESUME SYNC JOB `job_name`;
      1

    # Keywords

    RESUME, SYNC, LOAD

    # Best Practice

    PAUSE-SYNC-JOB STOP-SYNC-JOB
