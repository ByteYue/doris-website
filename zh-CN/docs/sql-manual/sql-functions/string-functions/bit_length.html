    bit_length

    # bit_length

    # description

    # Syntax

    INT bit_length(VARCHAR str)

    返回字符串的位长度。

    # example

    mysql> select bit_length("abc");
+-------------------+
| bit_length('abc') |
+-------------------+
|                24 |
+-------------------+

mysql> select bit_length("中国");
+----------------------+
| bit_length('中国')    |
+----------------------+
|                   48 |
+----------------------+
